A/B testing

We had tried to create familiarity for users and to test how users would interact with our new design before launching. For example, we had design new component, featured brands, which will suggest popular brands for user based on each segmentation. We did an experimentation based on this component by designing based on our old UI design and implemented it on upper funnel pages. Based on number of clicks result that we had gathered from UX Cam, we had found that the navigation with both logos and photos of the most popular items gained more interactions. As a result, we were able to move forward with the new component we had redesigned.

redesign

Before we have a homepage and upper funnel pages re-platforming, we gradually put some designs for A/B testing to test out how users interact with the design. For the re-platforming, we designed a new journey for users to find products more easily by using personalization— personalized segmentation based on the user’s browsing history. Personalization will be segmented into five groups excluding unknown users: beauty, women, men, mom and kids, and home. Each segmentation will reflect hero content, category navigations, and product recommendations.

wireframe

hero banner

We changed from promotions as a text form to promo blocks carousel where users can go to promotion/sale pages or collect coupon codes directly from the hero contents. After the re-platform, users will see hero banner based on their browsing history. For example, if a users browse mostly in Men, the hero content will change accordingly.

category navigation

As photos are easier to be recognized, category navigation has been changed from icons to generic photo or photo of a popular item for each category. After the re-platform, users will see category navigation based on their browsing history. For example, users who are in Beauty segment will see L2 and L3 from Beauty category.

coupons

We designed coupons based on printed coupon vouchers, so users can sense the same experience for both offline and online store.

product sku

Redesign for better information hierarchy: increasing product photo’s size and font size for product’s brand and price.

content design

We design contents for each five segmentations. Each user will see promotions ad products based on their shopping behaviors and searches.
